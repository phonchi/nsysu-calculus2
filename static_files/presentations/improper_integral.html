瑕積分與奇函數
∞ − ∞ ≠ 0

一個微積分中最常見的「直覺陷阱」完整解析

$$\int_{-\infty}^{+\infty} x\, dx = \;???$$
往下滑，看看為什麼↓
Part 01

直覺：奇函數在對稱區間積分為零？

我們都學過，如果 $f(x)$ 是奇函數（即 $f(-x) = -f(x)$）， 那麼在有限的對稱區間上：

$$\int_{-a}^{a} f(x)\, dx = 0 \quad (a \text{ 是有限值})$$

例如 $f(x) = x$ 是奇函數，$\displaystyle\int_{-3}^{3} x\, dx = 0$，完全正確。 但如果我們把範圍推到無窮大呢？

$$\int_{-\infty}^{+\infty} x\, dx \;\stackrel{?}{=}\; 0$$

答案是：不行！這個積分是發散的。 要理解為什麼，我們先來看瑕積分的嚴格定義。

Part 02

瑕積分的嚴格定義

定義

$\displaystyle\int_{-\infty}^{+\infty} f(x)\, dx$ 的定義是：

$$\int_{-\infty}^{+\infty} f(x)\, dx \;=\; \underbrace{\lim_{a \to \infty} \int_{-a}^{c} f(x)\, dx}_{\text{左半段}} \;+\; \underbrace{\lim_{b \to \infty} \int_{c}^{b} f(x)\, dx}_{\text{右半段}}$$

其中 $a$ 和 $b$ 是兩個獨立的變數，各自趨近 $\infty$。
整個積分收斂的條件是：兩個極限都必須各自存在且為有限值。
只要有任何一段的極限是 $\pm\infty$（不是有限值），整個積分就發散。

套用到 $\displaystyle\int_{-\infty}^{+\infty} x\, dx$

取 $c = 0$，分成左右兩段：

$$\text{右半段：}\quad \lim_{b \to \infty}\int_{0}^{b} x\, dx = \lim_{b \to \infty} \frac{b^2}{2} = +\infty \quad \text{（不是有限值 → 發散！）}$$
$$\text{左半段：}\quad \lim_{a \to \infty}\int_{-a}^{0} x\, dx = \lim_{a \to \infty} \left(-\frac{a^2}{2}\right) = -\infty \quad \text{（不是有限值 → 發散！）}$$

右半段的極限是 $+\infty$，不是有限值。根據定義，到這裡就結束了。

某一段極限 $= \pm\infty$
→
不是有限值
→
該段發散
→
整體發散

注意：發散的判定到此完成。不需要討論 $\infty - \infty$ 能不能抵消——根本走不到那一步，因為每一段各自就已經不是有限值了。

Part 03

為什麼 $a$ 和 $b$ 可以不同速度？

你可能會問：定義裡為什麼讓 $a$ 和 $b$ 各自獨立趨近 $\infty$？ 不能直接規定 $a = b$ 嗎？

💡 核心精神：「收斂」意味著結果不依賴取極限的方式

數學上，一個量要被稱為「收斂」，最基本的要求是： 不管你怎麼逼近，結果都必須是同一個值。

就好像你要確認一間餐廳好不好吃，不能只問一個人的意見—— 你要問很多不同的人，如果每個人都說好吃，那才是真的好吃。

同理，瑕積分讓 $a$ 和 $b$ 自由變動，就是在測試所有可能的逼近方式。 如果不管 $a$ 和 $b$ 怎麼跑（同速、不同速、一快一慢……）， 結果都收到同一個有限值，那才是真正的收斂。

通過測試 vs 未通過測試

先看一個通過測試的例子：$\displaystyle\int_{-\infty}^{+\infty} \frac{x}{(1+x^2)^2}\, dx$

$$\lim_{a \to \infty}\int_{-a}^{0} \frac{x}{(1+x^2)^2}\, dx = -\frac{1}{2}, \quad \lim_{b \to \infty}\int_{0}^{b} \frac{x}{(1+x^2)^2}\, dx = +\frac{1}{2}$$

每段各自收斂到有限值。所以不管 $a$ 和 $b$ 怎麼跑，左半段永遠趨近 $-\frac{1}{2}$， 右半段永遠趨近 $+\frac{1}{2}$，加起來永遠是 $0$。答案和 $a, b$ 的速度無關。

每段各自 → 有限值
→
$a, b$ 怎麼跑都一樣
→
通過測試 → 收斂

再看未通過測試的例子：$\displaystyle\int_{-\infty}^{+\infty} x\, dx$

$$\int_{-a}^{b} x\, dx = \frac{b^2 - a^2}{2}$$

$a$ 和 $b$ 的不同取法直接改變結果：

$a$ 和 $b$ 的關係積分值$a,b \to \infty$ 時
$a = b$（同速） $\dfrac{b^2 - b^2}{2} = 0$ $\to 0$
$b = 2a$（右邊跑兩倍快） $\dfrac{4a^2 - a^2}{2} = \dfrac{3a^2}{2}$ $\to +\infty$
$a = 2b$（左邊跑兩倍快） $\dfrac{b^2 - 4b^2}{2} = -\dfrac{3b^2}{2}$ $\to -\infty$
$b = a + 3$（右邊多 $3$） $3a + \dfrac{9}{2}$ $\to +\infty$

同速得 $0$，不同速得 $\pm\infty$。結果取決於 $a, b$ 的比例。

不同速度 → 不同結果
→
答案不唯一
→
未通過測試 → 發散

結論：讓 $a, b$ 可以不同速度，是為了全面測試積分值是否穩定。 真正收斂的積分，不管 $a, b$ 怎麼跑，結果都一樣。 如果只有特定速度（如 $a = b$）才能得到有限值， 那就代表這個積分的「答案」依賴於你的取法——這不是真正的收斂。

Part 04

對比：L'Hôpital 的 $\infty - \infty$ 為什麼不能直接說發散？

你在微積分裡也學過，$\infty - \infty$ 是 L'Hôpital 法則中的「不定式」之一。 遇到不定式時，我們不能直接下結論，而是要進一步化簡才能求值。 那為什麼瑕積分直接判發散，而 L'Hôpital 的不定式卻要繼續算呢？

💡 關鍵差異：一個變數 vs 兩個獨立變數

L'Hôpital 處理的是單一變數的極限。 $\infty - \infty$ 的兩部分被同一個變數綁定， 所以兩項之間的關係是固定的，有可能化簡後得到確定值。

瑕積分的 $\infty - \infty$ 來自兩個獨立的變數 $a$ 和 $b$。 兩段之間沒有任何關聯，所以結果不確定。

L'Hôpital 的例子：$\infty - \infty$ 但有確定答案

$$\lim_{x \to \infty} \left( x - \sqrt{x^2 - 1} \right)$$

$x \to \infty$ 時，$x \to \infty$ 且 $\sqrt{x^2-1} \to \infty$，形式上是 $\infty - \infty$。 但這裡只有一個變數 $x$——兩項的增長速度完全由 $x$ 決定， 沒有「不同速度」的選擇空間。有理化後：

$$\lim_{x \to \infty} \frac{x^2 - (x^2 - 1)}{x + \sqrt{x^2 - 1}} = \lim_{x \to \infty} \frac{1}{x + \sqrt{x^2 - 1}} = 0$$

不定式消失了，答案確定是 $0$。因為只有一個變數，兩項的關係是鎖死的。

再看一個例子

$$\lim_{x \to 0^+} \left( \frac{1}{x} - \frac{1}{\sin x} \right)$$

$x \to 0^+$ 時兩項都趨近 $+\infty$，形式上是 $\infty - \infty$。 通分後用 L'Hôpital 法則：

$$\lim_{x \to 0^+} \frac{\sin x - x}{x \sin x} \;\overset{\text{L'H}}{=}\; \lim_{x \to 0^+} \frac{\cos x - 1}{\sin x + x\cos x} \;\overset{\text{L'H}}{=}\; \lim_{x \to 0^+} \frac{-\sin x}{2\cos x - x\sin x} = \frac{0}{2} = 0$$

同樣只有一個變數 $x$，兩項被綁在一起，化簡後得到確定值。

為什麼瑕積分不能這樣做？

$$\underbrace{\lim_{a \to \infty}\left(-\frac{a^2}{2}\right)}_{-\infty} \;+\; \underbrace{\lim_{b \to \infty} \frac{b^2}{2}}_{+\infty}$$

這裡有兩個獨立變數。不像 L'Hôpital 的例子中兩項都被同一個 $x$ 控制， 這裡 $a$ 和 $b$ 之間沒有任何關係。 你無法通分、無法有理化、無法用 L'Hôpital——因為根本沒有一個共同的變數可以操作。

✅ L'Hôpital 的 $\infty - \infty$

單一變數控制兩項

兩項的速度被綁定

可以化簡 → 可能有確定答案

→ 不定式，要繼續算

❌ 瑕積分的 $\infty - \infty$

兩個獨立變數各自趨近 $\infty$

兩段的速度不綁定

無法化簡 → 結果隨取法而變

→ 未通過測試，直接判發散

Part 05

互動實驗：自己試試看

下面你可以自由調整 $a$ 和 $b$，模擬「用不同速度趨近 $\infty$」的效果。 如果積分真的收斂，那不管你怎麼調，結果都應該一樣。

▶ 拖動滑桿調整端點

a = 10
左端點 $-a$：$a$ 越大，左邊跑越遠
b = 10
右端點 $+b$：$b$ 越大，右邊跑越遠

動手試試：先設 $a = b = 10$（結果 $= 0$）。然後只動 $b$，拉到 $20$—— 結果瞬間從 $0$ 跳到 $150$。 如果積分真的收斂，不管怎麼調都應該一樣才對，但事實上不同的 $a, b$ 給出了完全不同的結果。 測試沒通過——發散。

Part 06

什麼時候奇函數積分才真的是零？

前提只有一個：瑕積分本身要收斂（每段極限都是有限值）。

例如 $\displaystyle f(x) = \frac{x}{(1 + x^2)^2}$ 是奇函數，而且：

$$\int_{0}^{+\infty} \frac{x}{(1+x^2)^2}\, dx = \frac{1}{2} \quad \text{（有限值 → 收斂！）}$$

每段各自收斂到有限值。不管 $a, b$ 怎麼跑，左半段永遠趨近 $-\frac{1}{2}$， 右半段永遠趨近 $+\frac{1}{2}$，加起來永遠是 $0$。 所有速度都給出同一個答案——通過測試，收斂。

$$\int_{-\infty}^{+\infty} \frac{x}{(1+x^2)^2}\, dx = -\frac{1}{2} + \frac{1}{2} = 0 \;\; \checkmark$$

Part 07

整理：完整的邏輯鏈

邏輯鏈

第一層（定義）：直接判定發散

瑕積分拆成兩段，各自取極限。
$\displaystyle\int_0^{+\infty} x\, dx = +\infty$，不是有限值 → 該段發散 → 整體發散。結案。

第二層（動機）：為什麼定義讓 $a, b$ 獨立？

因為「收斂」的意思是不管怎麼逼近，結果都一樣。
讓 $a, b$ 自由變動，是在全面測試結果是否穩定。
$\displaystyle\int_{-a}^{b} x\, dx$ 隨 $a, b$ 比例而變 → 測試沒通過 → 發散。

對比 L'Hôpital

L'Hôpital 的 $\infty - \infty$ 只有單一變數，兩項速度被綁定，有機會化簡出確定值。
瑕積分的兩段由獨立變數控制，速度不綁定 → 結果不確定 → 直接判發散。

📌 總結

奇函數在對稱區間積分為零的前提是積分本身收斂。
「收斂」的核心要求：不管 $a, b$ 以什麼速度趨近 $\infty$，結果都必須相同。
$\displaystyle\int_{-a}^{b} x\, dx = \frac{b^2 - a^2}{2}$，結果隨 $a, b$ 比例而變 → 發散。
對比 L'Hôpital：單一變數的 $\infty - \infty$ 兩項速度綁定，有機會化簡，不能直接說發散。
